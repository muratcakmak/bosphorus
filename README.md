# bosphorus

Component Library

## Installation

```sh
yarn install bosphorus
```

### Web

```package.json
  "alias": {
    "react-native": "react-native-web"
  }
```

## Usage

```js
import { Button } from "bosphorus";

// ...
<Button title="Press" />
```
